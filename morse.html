EMG·MORSE
Accessibility tool. This page types text using only eye blinks, decoded from an EOG (electrooculography) signal into Morse code in real time -- no hands, no keyboard. It's built for anyone with limited or no hand/arm mobility (e.g. ALS, spinal cord injury, or other motor-control conditions) where a blink can remain one of the few reliably repeatable voluntary movements available.
